# **App Name**: LinkUp9ja

## Core Features:

- Profile Creation: Users create detailed profiles with photos, bios, interests, and location information.
- Swipe-Based Matching: Users swipe through profiles and indicate interest, with mutual likes resulting in a match.
- Real-time Chat: Matched users can communicate via real-time chat with text and media sharing capabilities.
- Preference Setting: AI-powered suggestion tool, suggest preferences of a user during onboarding
- Notifications: Users receive notifications for new matches and messages.
- Safety Features: Users can report or block other users. Photos are verified for authenticity. User are also automatically flagged if AI detects harassment.
- Premium Subscription: Offer premium features such as unlimited swipes and profile boosts via a subscription.

## Style Guidelines:

- Primary color: Coral Pink (#FF6B6B) to convey a modern, soft, and welcoming feel.
- Background color: Light Gray (#F8F9FA) for a clean and unobtrusive backdrop.
- Accent color: Gold (#F1C40F) for highlights and calls to action, providing a touch of luxury.
- Headline font: 'Poppins' sans-serif font for headings, providing a modern and clean aesthetic.
- Body font: 'Inter' sans-serif font for body text, ensuring readability and a contemporary look.
- Use modern and minimalist icons for clear navigation and visual appeal.
- Card-based layouts with rounded corners (12-16px) and glassmorphism effects to create a soft and engaging interface.
- Smooth animations and transitions (300-400ms) to enhance user experience and engagement.
